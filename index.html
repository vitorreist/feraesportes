🔥 Firebase Config
Configure uma vez — sincroniza e autentica automaticamente em todos os dispositivos.
Como configurar:
1. console.firebase.google.com → projeto feraesportes
2. Realtime Database → copie a URL
3. Regras do banco → cole:
  {"rules":{"fera_db":{".read":"auth!=null",".write":"auth!=null"},"roles":{".read":"auth!=null",".write":false}}}
4. Configurações do projeto ⚙️ → Seus apps → Web → copie a apiKey
5. Authentication → Sign-in method → ativar E-mail/senha
6. Authentication → Users → Adicionar usuário
7. Realtime Database → Dados → criar nó roles/{uid} = "admin" ou "operador"
⚙️ Setup Inicial — Papéis de Usuário
Clique para gravar os papéis dos usuários no banco. Faça isso uma única vez após configurar o Firebase.
